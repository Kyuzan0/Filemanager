File Manager - Bulk Operations Test Suite

Halaman ini berisi kumpulan test untuk memverifikasi fungsi bulk operations pada File Manager.

1. Test Select Multiple Items

Test fungsi select multiple items di File Manager interface.

Yang diuji:

  1. Checkbox individual untuk memilih item
  2. Checkbox "Select All" untuk memilih semua item
  3. State indeterminate untuk partial selection
  4. Update tombol action berdasarkan jumlah item terpilih
Buka Test Select Multiple Buka File Manager

2. Test Bulk Delete

Test fungsi delete multiple items dan logging-nya.

Yang diuji:

  1. Delete multiple items dalam satu request
  2. Verifikasi bulk logging di activity log
  3. Display informasi bulk action dengan jelas
Catatan: Pastikan sudah ada file/folder untuk dihapus. Gunakan test ini untuk membuat file test terlebih dahulu.
Buka Test Bulk Delete Lihat Activity Log

3. Test Bulk Upload

Test fungsi upload multiple files dan logging-nya.

Yang diuji:

  1. Upload multiple files dalam satu request
  2. Verifikasi bulk logging untuk upload
  3. Penanganan file besar vs file kecil
Catatan: Upload minimal 2 file untuk test bulk upload. File kecil (<5MB) akan diupload bersamaan, file besar akan diupload individual.
Buka Test Bulk Upload Lihat Activity Log
